Global Tuberculosis Trends Analysis

This project analyzes global tuberculosis (TB) trends using public health data from Our World in Data (OWID) and the World Health Organization (WHO). It focuses on cleaning large datasets, analyzing TB incidence, mortality, and treatment success rates, and creating clear visualizations to compare trends across countries and over time.

The project demonstrates a complete, reproducible Python data science workflow, including data loading, cleaning, analysis, and visualization.

Key skills demonstrated:

Data cleaning and preprocessing

Public health data analysis

Dataset merging and comparison

Data visualization and reporting


## Tools - Python - Pandas - Plotly - Matplotlib - VS Code
